This directory contains a modified version of Tcl 7.4.  The modifications
are as follows:

* Removed documentation.

* Removed use of sprintf() (not finished yet).

* Removed use of mktemp().

* Removed losts of code amd #ifdefs for compatibility with non-POSIX systems.

* Removed sccs line at the top of each file.

* Converted prototypes to ANSI C (not finished yet).
