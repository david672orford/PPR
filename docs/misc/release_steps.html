PPR Release Steps

These are the steps required to release a new version of ppr:

  1. Make sure the version and date are correct in these places:
    • top level source directory name
    • include/version.h
    • include/version.mk
    • History.txt
    • docs/misc/docs_history.txt
    • docs/misc/announce.txt
    • docs/misc/ppr.lsm
    • ppr-X.XX.spec
  2. Upload to ftp://ppr-dist.trincoll.edu/pub/ppr/.
  3. Update http//ppr.trincoll.edu/download.html.
  4. Upload to ftp://upload.sourceforge.net/incoming/.
  5. Create the release notes in announce.txt. These will be a summary of the changes recorded in History.txt.
  6. Create a release on Sourceforge.
  7. Send an announcement to ppr-list@mail.trincoll.edu.
  8. Post an announcement on Freshmeat with abbreviated release notes.
  9. Post news on Sourceforge with the same abbreviated release notes.
  10. Take a deap breath.
